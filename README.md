# GitHub Live Portfolio (Persian RTL)

این پروژه دیگر محتوای فیک ندارد و پورتفولیو را مستقیم از GitHub API می‌سازد.

## اجرا

```bash
python3 -m http.server 5173
```

بعد مرورگر را باز کنید:

- `http://localhost:5173` (وارد کردن یوزرنیم)
- `http://localhost:5173/?u=YOUR_GITHUB_USERNAME` (لود خودکار)

## چه چیزهایی نمایش داده می‌شود؟

- اطلاعات پروفایل (نام، بیو، فالوئر، تعداد ریپو)
- بهترین ریپوها بر اساس Star
- زبان‌های پرتکرار در ریپوها

## نکته

اگر با محدودیت GitHub API روبه‌رو شدید، چند دقیقه بعد دوباره تست کنید.

# Personal Portfolio (Persian RTL)

یک سایت پورتفولیوی شخصی، تمیز و زیبا با طراحی RTL.

## Run locally

```bash
python3 -m http.server 8000
```

Open:

- `http://127.0.0.1:8000`

## Customize

- اطلاعات تماس در بخش Contact داخل `index.html`
- مهارت‌ها و پروژه‌ها در همان فایل
- رنگ‌بندی و استایل در `styles.css`
